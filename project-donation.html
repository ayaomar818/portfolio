Donation Popup System

WooCommerce Integrated Donation Flow

Project Overview

A custom-built donation system designed to provide a smooth and flexible donation experience. Users can choose between one-time or monthly donations with dynamic payment options.

How It Works

The popup is triggered dynamically when a user clicks a donation button on a specific post. The system retrieves post data and displays it inside the popup.

  • Left section: Displays dynamic campaign information
  • Right section: Donation type selection (one-time / monthly)
  • Custom amount input with predefined options
  • Optional commission toggle
  • Checkout integration with WooCommerce

Key Features

  • Dynamic content loading from posts
  • Conditional UI logic based on donation type
  • Custom payment handling
  • WooCommerce checkout integration
  • Optimized UX flow

Technologies Used

WordPress, WooCommerce, PHP, JavaScript, AJAX
